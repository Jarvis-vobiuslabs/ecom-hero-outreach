# Ecom Hero Outreach Review (Skincare/Cosmetics)

Phone-friendly review app for outbound leads + personalized email drafts.

## Data
- `data/leads.json` — source of truth

## View
GitHub Pages serves the app from repo root.

## Workflow
1) Add leads to `data/leads.json` (manually or via scripts)
2) Review on phone
3) Copy email draft and send manually from Gmail/IONOS

No auto-send.
